How you use it

1

Import and mount from your real container

import { snapshotBones, mountSkeleton } from './src/index.js'

const root = document.getElementById('card')
const data = snapshotBones(root, 'card')
mountSkeleton(document.getElementById('skeleton'), data)

How it works

  1. Render your UI; Tailwind styles the live DOM as usual.
  2. Snapshot walks visible nodes and stores x/w (%) and y/h (px), plus radius r. Optional c = container bone.
  3. Render outputs absolutely positioned divs with Tailwind (animate-pulse, arbitrary left-[…%], etc.).

Install

Clone or copy this repo, install dev deps (for the esbuild bundle), and serve over HTTP (ES modules will not load from file://). 

cd skeleton-loading
npm install
npx serve .
# or open via Apache / XAMPP → demo/index.html

npm run build:iife
# → dist/boneyard-tailwind.min.js (global BoneyardTailwind)

Output

JSON bundles optional breakpoints (width key → snapshot). Each snapshot includes bones[] with { x, y, w, h, r, c? }.
